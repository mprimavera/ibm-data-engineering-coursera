## Overview

This repository contains the work completed throughout the IBM Data Engineering Professional Certificate program on Coursera. The program focuses on building practical skills in data engineering, including:

- Data extraction, transformation, and loading (ETL)
- Relational and big data storage
- Cloud-based data workflows
- Analytics-ready data pipelines

## Course Modules

1. **Python Basics** – Data types, loops, functions, and basic scripting.
2. **SQL & Databases** – Writing queries, joins, aggregations, and working with relational databases.
3. **ETL Pipelines** – Extracting, transforming, and loading data efficiently.
4. **Data Warehousing** – Designing schemas, building pipelines, and analytics-ready storage.
5. **Big Data** – Using Spark and Hadoop for large-scale data processing.
6. **Cloud & DevOps Basics** – Deploying data pipelines and managing workflows on IBM Cloud.

> Each module folder contains additional README files and scripts related to the exercises and mini-projects completed.
